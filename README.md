# srqlock

A non-flash based version of the Uniqlo Uniqlock utilizing Giphy gifs instead of dancing videos.

A deployed instance of this project can be found [here](https://samrum.com). The original Uniqlock has been shutdown, but a video of how it looked can be found [here](https://www.youtube.com/watch?v=rAMC1NizEU0).
